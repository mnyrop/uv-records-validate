# uv-records-validate

## Prerequisites
- Pyenv
- Git

## Setup

1. Install/update python version & upgrade pip
  ``` sh
  pyenv install --skip-existing
  pip install pipenv
  ```

2. Install Pipenv
  ``` sh
  pip install --upgrade pip
  ```

3. Install dependencies from Pipfile
  ``` sh
  pipenv install
  ```

## Run

1. Launch the jupyter notebook
  ``` sh
  pipenv run jupyter notebook
  ```
